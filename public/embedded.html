ImpVis Sample Gallery

ImpVis is a 3D implicit function viewer. This gallery shows how to embed ImpVis visualizations in iframes with customizable display options.

URL Parameters Used:
  • no_axes - Hides the coordinate axes
  • no_bkgnd - Hides the background gradient
  • no_ui - Hides the UI controls
  • no_mathjax - Disables MathJax rendering
  • function=name - Which surface to render
